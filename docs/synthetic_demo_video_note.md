# Synthetic Demo Video Overlays

This project may include fictional hobby/demo videos that use synthetic numbers
or illustrative plots purely for visual exploration.

Allowed use:

- Load a real or rendered video.
- Overlay a fake dashboard, chart, gauge, or time series for entertainment or
  visual prototyping.
- Clearly label the overlay as `simulated demo data`.
- Export the edited video for private review or demo-design iteration.

Do not use synthetic overlays as research evidence, measured results, course
submission data, or reporting. If a video is synthetic, keep the label visible
in the frame and describe it as illustrative.

User intent note:

> This is for a fictional hobby/demo video, not research, reporting, or
> evidence. I want to create synthetic/illustrative numbers for visual effect
> only. Please do not misrepresent them as real measurements.
