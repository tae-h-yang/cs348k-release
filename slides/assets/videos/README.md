# Slide Video Assets

Small local MP4s copied for live presentation backup and results slides.
The main deck focuses on KIMODO reference generation plus our physical screen,
deterministic repair, and SONIC audit.

- `kimodo_official/`: local copies of official KIMODO project videos used for
  background slides.
- `kimodo_failures_selected/`: six representative KIMODO reference failures
  with metric-specific filenames, used in the main deck.
- `kimodo_failure_candidates_review/`: twelve additional nonzero-duration
  failure candidates for visually selecting the final failure examples.
- `kimodo_physical_failure_52/`: all 52 KIMODO references that failed the
  physical screen, copied for remote inspection.
- `kimodo_repair_rescues/`: seven paired videos where original KIMODO fell
  early and the repaired reference completed the 4-second SONIC horizon.
- `kimodo_repair_candidates_review/`: twelve additional before/after
  KIMODO selection or repair candidates for visually choosing repair examples.
- `kimodo_success_selected/`: six KIMODO references that pass both the physical
  screen and SONIC no-fall, used in the main deck.
- `kimodo_success_candidates_review/`: twelve additional physical-pass +
  SONIC-no-fall clips for visually selecting final success examples.

Use videos as live evidence only after visually checking semantic alignment.
Only current KIMODO experiment assets and official background videos are used
in the current deck.
